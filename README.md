# IB_Hospital

## Description
Site web statique pour la gestion et la présentation des services d'un hôpital (IB Hospital). Contient des pages pour chirurgie, maternité, médecine générale, urgences et pages de connexion.

## Structure du projet
- `index.html` — page d'accueil
- `style.css` — styles globaux
- `chirurgie/chirurgie.html` — page chirurgie
- `maternité/maternite-pediatrie.html` — page maternité et pédiatrie
- `medecine generale/medecine-generale.html` — page médecine générale
- `urgence/Urgence.html` — page urgences
- `login_connexion/connexion.html` — page de connexion
- `login_connexion/registre.html` — page d'inscription
- `image/` — ressources images

## Lancer le projet
1. Ouvrir `index.html` dans un navigateur web (double-clic ou "Ouvrir avec").
2. Aucune dépendance serveur requise — site statique.

## Modifier ou développer
- Éditez les fichiers HTML et `style.css` dans un éditeur de texte.
- Pour tests locaux avancés, servez le dossier avec un petit serveur HTTP :

  - Python 3 : `python -m http.server 8000`
  - Node.js (http-server) : `npx http-server -p 8000`

Puis ouvrez `http://localhost:8000`.
